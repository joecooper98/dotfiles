# vimrc


git clone this repo, and then run
```
./install.sh
```
or
```
cp -rf vimrc/.vim ~/.vim
```
```
cp -rf vimrc/.vimrc ~/.vimrc
```
After that, open up vim, and run 

:PluginInstall

Then you have to install COC by

:CoCinstall

It should then work!

use the following packages

zsh/ohmyzsh/subnixr minimal

kitty with nord theme

spotifytui
